<h1 align="center">The <em>.github</em> Repository</h1>

This repository provides organisation-wide community resources and default GitHub configuration for repositories within this organisation.

It may contain the organisation profile, issue templates, pull request templates and other default community health files that GitHub automatically applies where appropriate.

Project source code, reusable workflows, repository templates and other shared resources are maintained in their own dedicated repositories.
