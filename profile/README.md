<h1 align="center">The Lupaxa Project Private</h1>

**The Lupaxa Project Private** is the home of the internal tooling and infrastructure that supports **The Lupaxa Project**.

While the public organisations host open-source software, reusable workflows and community resources, this organisation focuses on the tools used to develop, maintain and operate the wider ecosystem.

## Purpose

Repositories within this organisation may include:

- GitHub organisation management tools
- Repository administration utilities
- Automation for labels, milestones and projects
- Internal workflows and operational tooling
- Infrastructure and maintenance utilities
- Supporting services and developer tooling

These repositories are primarily intended for project maintainers and are not generally published as open source.

## Relationship to The Lupaxa Project

This organisation forms part of **The Lupaxa Project** ecosystem and exists to support its day-to-day operation.

Where appropriate, tooling developed here may later be released through one of the public organisations once it is suitable for general use.
